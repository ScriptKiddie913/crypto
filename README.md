## üîç Overview

SOTANIK Forensic Hub is an enterprise-grade blockchain forensics platform designed for security researchers, investigators, and compliance professionals. The platform provides real-time visualization and analysis of Bitcoin and Ethereum blockchain transactions, enabling comprehensive investigation of wallet addresses, transactions, and block data across multiple networks.

## ‚ú® Core Features

### Multi-Chain Support
- **Bitcoin (BTC)** - Full mainnet transaction tracking
- **Ethereum (ETH)** - Mainnet and Sepolia testnet support
- Cross-chain correlation and analysis

### Advanced Visualization
- **Interactive Network Graphs** - D3.js-powered force-directed graphs
- **Real-time Rendering** - Dynamic node and link updates
- **Zoom & Pan Controls** - Infinite canvas navigation
- **Color-Coded Entities** - Visual distinction between address types

- **Deep Transaction Tracing** - Multi-level recursive expansion
- **Risk Assessment** - Automated risk scoring algorithm
- **Entity Recognition** - Pre-configured entity registry (exchanges, known wallets)
- **OSINT Integration** - Direct links to blockchain explorers

### Investigation Tools
- **PDF Report Generation** - Professional dossier export with jsPDF
- **Node Selection & Inspection** - Detailed metadata panels
- **Transaction Flow Analysis** - Input/output relationship mapping
- **Block Explorer Integration** - Mempool, Blockstream, Etherscan, Blockscout
